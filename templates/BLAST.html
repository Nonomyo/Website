Basic Local Alignment Search Tool

De Basic Local Alignment Search Tool, ook bekend als BLAST, is een tool om een gegeven sequentie (DNA, RNA, eiwit) te vergelijken met bekende sequenties uit de NCBI database. Na een BLAST krijgt de gebruiker een selectie sequenties terug die vergelijkbaar zijn met de opgegeven sequentie.

Er zijn enkele belangrijke opties/instellingen voor het goed uitvoeren van een BLAST. Ten eerste moet BLAST weten met wat voor sequentie het werkt (nucleotiden of aminozuren). Daarnaast heb je in BLAST enkele opties voor hoe je de sequentie meegeeft, dit kan bijvoorbeeld een accession number of een FASTA-bestand zijn.

Zodra een sequentie is opgegeven voor een BLAST, duurt het vaak een tijdje voordat de BLAST klaar is vanwege het grote aantal sequenties in de NCBI-database. Zodra de BLAST klaar is, wordt er een lijst aan sequenties gegeven die tot een bepaald niveau op de opgegeven sequentie lijken.
