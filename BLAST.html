Basic Local Alignment Search Tool

De Basic Local Alignment Search Tool, ook bekend als BLAST, is een tool om een gegeven sequentie (DNA, RNA, eiwit) te vergelijken met bekende sequenties uit de NCBI database. Na een BLAST krijgt de gebruiker een selectie sequenties terug die vergelijkbaar zijn met de opgegeven sequentie.

Er zijn enkele belangrijke opties/instellingen voor het goed uitvoeren van een BLAST. Ten eerste moet BLAST weten met wat voor sequentie het werkt (nucleotiden of aminozuren). daarnaast heb je in BLAST enkele opties voor hoe je de sequentie meegeeft, dit kan bijvoorbeeld een accasion number of een fasta file zijn.

Zodra een sequentie is opgegeven voor een BLAST, duurt het vaak een tijdje voordat de BLAST klaar is vanwege het grootte aantal sequenties in de ncbi database. Zodra de BLAST klaar is, worden er een lijst aan sequenties gegeven die tot een bepaal niveau op de gegeven sequentie lijken.
